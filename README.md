# Figma Prototype Portfolio Design

A portfolio profile design prototype crafted in Figma, intended for personal branding and professional presentation. This project provides a downloadable PDF of the design, offering a clear structure and modern aesthetic for portfolio sites.

## Contents

- **portfolio-profile-design.pdf:** The main Figma design export.
- **README.md:** Project summary and usage instructions.

## Features

- Clean, modern layout for portfolio profiles.
- Ready-to-use design for web or print.
- Useful as a reference or template for personal branding.

## Usage

1. Clone or download this repository:
   ```sh
   git clone https://github.com/natnaeleyuel/UGR-4424-15-figma-prototype-portfolio-design.git
   ```
2. Open `portfolio-profile-design.pdf` to view the prototype.
3. Use the design as inspiration or adapt it for your own portfolio.
